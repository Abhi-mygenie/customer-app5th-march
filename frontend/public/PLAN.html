PLAN-2026-01 — Atlas Resilience Hardening

Parent: INV-2026-01-prod-logs-analysis Target: Customer App backend Risk: High Est. effort: 2.5 h wall-clock post-approval Fast Lane: NOT eligible
Purpose
Make the Customer App survive a MongoDB Atlas primary loss without returning 500 to end users. Two-tier fix: a connection-string change (env-only, 1 line, immediate) and a small resilient wrapper around GET /api/config/{restaurant_id} (~60 LOC, additive). No frontend changes, no schema changes, no dependency changes.

C2 — Resilient config endpoint + LRU cache High risk

Sketch (pseudo — not code):

_CONFIG_CACHE: Dict[str, Tuple[dict, float]] = {}
_CONFIG_CACHE_TTL_SECONDS = 60

def _default_config_block(restaurant_id: str) -> dict:
    # extracted verbatim from current lines 993-1104
    return { "restaurant_id": restaurant_id, "showLogo": True, ... }

async def _get_config_resilient(restaurant_id: str) -> Tuple[dict, str]:
    """Returns (config, source) where source ∈ {'db','cache','default'}."""
    now = time.monotonic()
    try:
        cfg = await db.customer_app_config.find_one({"restaurant_id": restaurant_id}, {"_id": 0})
        if cfg is None:
            cfg = _default_config_block(restaurant_id)
        _CONFIG_CACHE[restaurant_id] = (cfg, now)
        return cfg, "db"
    except PyMongoError as e:
        logging.warning(f"[atlas-resilience] Mongo error restaurant_id={restaurant_id}: {type(e).__name__}")

    if restaurant_id in _CONFIG_CACHE:
        cached, cached_at = _CONFIG_CACHE[restaurant_id]
        if now - cached_at < _CONFIG_CACHE_TTL_SECONDS:
            return cached, "cache"

    return _default_config_block(restaurant_id), "default"

@config_router.get("/{restaurant_id}")
async def get_app_config(restaurant_id: str, response: Response):
    cfg, source = await _get_config_resilient(restaurant_id)
    response.headers["X-Config-Source"] = source
    return cfg

C3 — Health endpoints Low risk

@app.get("/api/health/live")
async def health_live():
    return {"status": "ok"}

@app.get("/api/health/ready")
async def health_ready():
    try:
        await asyncio.wait_for(client.admin.command("ping"), timeout=2.0)
        return {"status": "ok", "db": "reachable"}
    except Exception as e:
        return JSONResponse(status_code=503,
            content={"status": "degraded", "db": "unreachable", "reason": type(e).__name__})
